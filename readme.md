# CarePoint — Hospital Management System

A full-stack hospital management system built with Flask and PostgreSQL.

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file (see `.env.example`)
4. Create a PostgreSQL database called `carepoint`
5. Run: `python app.py`
